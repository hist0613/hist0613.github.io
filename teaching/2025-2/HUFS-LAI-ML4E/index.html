강의 개요

영어학 전공자를 위한 실용적인 기계학습 강의입니다. 복잡한 수학 없이도 영어 텍스트 분석 도구를 활용할 수 있도록 하는 것이 목표입니다. Python과 다양한 라이브러리를 통해 실습 위주로 진행합니다.

학습 목표

  • 영어 텍스트 데이터 처리 기초 기술 습득
  • 감정분석, 텍스트 분류 등 실용적 응용
  • 영어 문서 검색과 정보 추출
  • 최신 NLP 도구와 모델 활용법 이해
강의 정보
  • 강의 시간: 화요일 1,2,3교시 (09:00-12:00)
  • 강의실: 대학원6503
  • 학점: 3학점
  • 수강권장 사전지식: Python 기초, 선형대수, 확률·통계 기초
사용 도구
  • 언어: Python 3.11+
  • 라이브러리: NumPy, Pandas, Scikit-learn, NLTK
  • 딥러닝: PyTorch
  • 환경: Google Colab, Jupyter Notebook
