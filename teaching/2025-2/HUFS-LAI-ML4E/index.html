강의 개요

영어 텍스트 분석에 특화된 기계학습 기법을 학습합니다. 자연어처리의 기초부터 최신 트랜스포머 모델까지, 영어 데이터를 활용한 실무 중심의 교육을 제공합니다.

학습 목표

  • 영어 텍스트 전처리 및 특성 추출 기법 습득
  • 자연어처리를 위한 기계학습 알고리즘 이해
  • 영어 텍스트 분류, 감정 분석, 기계 번역 모델 구현
  • 트랜스포머 기반 모델(BERT, GPT) 활용법 학습
강의 정보
  • 강의 시간: 화요일 3,4,5교시 (11:00-14:00)
  • 강의실: 교수학습개발원 108호 (PC실습실)
  • 학점: 3학점
  • 수강권장 사전지식: Python 기초, 기계학습의이해, 영어 독해
사용 도구
  • 언어: Python 3.11+
  • 라이브러리: NLTK, spaCy, Transformers
  • 딥러닝: PyTorch, Hugging Face
  • 환경: Google Colab, Jupyter Notebook
